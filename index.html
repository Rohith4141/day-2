JavaScript library for building user interfaces (UIs). It allows developers to create reusable components.

Vue.js

A front-end framework that allows developers to create web apps using HTML, CSS, and JavaScript.

Languages used to create the visual aspects of a website, such as the graphical user interface (GUI), text, images, and videos

A Python-based web development framework that provides tools and libraries for building web apps. It's primarily used for backend development.

Ruby on Rails

A backend framework for web applications that's known for efficiency and scalability.
